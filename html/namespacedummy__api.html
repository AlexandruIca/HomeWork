std::vector<std::unique_ptr<dummy_api::Shape>>& dummy_api::get_anon_shapes()
#include <include/hwapi.hpp>

Accesses the vector of anonymous Shapes.

This vector holds pointers to shapes that are in the vector returned by get_shapes so that when you create a shape like this:

{
    Rectangle r{};
    // do stuff
} // scope ending

the destructor of the shape doesn't get called and there is no segfault. Call methods starting with lowercase to create anonymous shapes. The disadvantage of anonymous shapes is that you can't change anything about them later.

void dummy_api::draw_shapes()
#include <include/hwapi.hpp>

Draws all the shapes currently requested.

Should be called every frame since shapes may be updated frequently (eg. changes of color).

Warning

Does NOTHING else, only draws the shapes(no window event handling).

int dummy_api::draw(std::function<void(double)> t_call = [](double) -> void {})
#include <include/hwapi.hpp>

Updates the window and handles events.

Parameters
t_call in needs to be a void function that takes a float as a parameter. The parameter of that function is the elapsed time since the last frame.

This is an entire "main loop" that draws everything on the screen and stops when either the window is closed by the user or the 'ESC' key is pressed.
