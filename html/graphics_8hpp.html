◆ WITH

#define WITH (   ...)    [&](const double elapsed_time)

Runs code inside main loop.

If you want to modify the color of an object of type triangle let's say, then you can use this macro:

#include "graphics.hpp"
int main()
{
Triangle t{ 20, 20, 400, 400, 150, 100, RED };
return draw(WITH{
++(t.color().g);
t.color().g = t.color().g % 255;
});
}

which is here only to help you. You can just pass a lambda that takes a paramater of type double which is the elapsed time since last frame.
