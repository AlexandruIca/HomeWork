# Primitive
Libraria poate desena cateva figuri geometrice simple:
* puncte
* linii
* triunghiuri
* cercuri
* dreptunghiuri

Un program foarte simplu care foloseste libraria arata cam asa:
```c++
#include "graphics.hpp"

int main()
{
    point(50, 60);
    return draw();
}
```
Programul va desena un punct la coordonata (50, 60).
