Detailed Description

Every object that has information about a primitive is derived from Shape.

Cannot be instantiated(only through derived ojects). Instead of calling functions like point or line you can create objects that store information needed to draw the primitives so you can modify it later. Objects have the UpperCamelCase naming convention(eg. Line, Point). Constructors of derived objects should have the same parameters as their functions relatives. Each derived class will provide equality operators and comparison operators for the same types. Each derived class will provide methods for retrieving information like color and position.
