◆ draw()

int dummy_api::draw ( std::function< void()>  t_call_first = []() -> void {},
std::function< void()>  t_call_last = []() -> void {} 
)

Updates the window and handles events.

This is an entire "main loop" that draws everything on the screen and stops when either the window is closed by the user or the 'ESC' key is pressed.

Parameters
[in]t_call_firstneeds to be a void function with no arguments and will be called before the draw_shapes call inside draw.
[in]t_call_lastthe only difference between this and t_call_first is that it will be called after draw_shapes.
Return values
intReturns 0 to make it easy to write something like:
return draw();
in main.

◆ draw_shapes()

void dummy_api::draw_shapes ( )

Draws all the shapes currently requested.

Should be called every frame since shapes may be updated frequently (eg. changes of color).

Warning
Does NOTHING else, only draws the shapes(no window event handling).
